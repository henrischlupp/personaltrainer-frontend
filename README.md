# Personal Trainer Frontend

A React + TypeScript + Vite application for managing personal trainer customers and training sessions.

## Features

- View customer and training lists
- Search, sort and filter table data
- Add, edit and delete customers
- Add and delete training sessions
- Export customer data to a CSV file
- Deployed to Vercel

## Deployment

The app is deployed at:

https://personaltrainer-theta-liard.vercel.app/

## Tech Stack

- React
- TypeScript
- Vite
- Material UI
- MUI X Data Grid
- date-fns
- Vercel
